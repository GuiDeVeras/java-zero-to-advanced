Exercise 4
	Write a program to read an integer N and then the data (id, name, and salary) of N employees. There must be no duplicate IDs.
	Next, increase the salary of a specific employee by X percent. To do this, the program must read an ID and the value X. If the entered ID does not exist, display a message and abort the operation. At the end, show the updated list of employees, as shown in the examples.
	Remember to apply the encapsulation technique so that the salary cannot be changed freely. A salary can only be increased through a given percentage increase operation.
