Exercise 1
	Make a program that register a bank account, it's necessary the account's number, the account' holder and the initial deposit value. That inital deposit is optional, so if the holder don't choose to deposit in the registration, the balance will be zero.
	Once a bank accout got register, the account's number can not be changed. However the holder's name can be altered.
	The account's balance can not be changed freely. The balance only increase with deposit and decrease with withdraw. For each withdraw, the bank charges a fee of $ 5,00.
